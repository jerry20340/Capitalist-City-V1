# 🌆 資本都市 (Capital City) V24.3

《資本都市》是一款基於 HTML5 與 JavaScript 開發的網頁版**房地產策略模擬遊戲**。玩家將扮演投資者「KAI」，在 10x10 的城市網格中，透過收購、合併土地、改建大樓以及產權鎖定，與其他 14 位家族競爭，爭奪城市資產霸主的地位。

## 🎮 遊戲核心機制

### 1. 土地分區與容積率 (Zoning & Floor Area Ratio)
本遊戲的建築高度上限（Max Floor）由**「臨路數量 (Road Access)」**決定。地塊四周鄰近道路的數量越多，開發潛力越高：

| 土地類型 | 0 路 | 1 路 | 2 路 | 3 路 | 4 路 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **工業區 (I)** | 1F | 2F | 2F | 2F | 2F |
| **住宅區 (R)** | 3F | 7F | 9F | 9F (可突破) | 2F |
| **商業區 (C)** | 2F | 7F | 9F | 9F (可突破) | 2F |

* **開發突破**：若地塊位於 **3 面環路**，且透過「土地合併 (Merge)」機制與「重新建築 (Rebuild)」，最高可突破興建至 **12 層樓**。

### 2. 收益與租賃邏輯 (Revenue Model)
收益計算採每回合結算，受樓層位置與租客類型影響：

* **樓層加成**：
    * **1F 店面**：收益增加 **50%**。
    * **頂樓加成**：4F 以上建築之最高樓層，收益增加 **20%**。
* **租客機制**：每回合有 20% 機率吸引品牌進駐（固定租約 5 年），有租客時該層收益增加 **15%**。
* **維護成本**：每層樓需支付固定維護費。若產權處於「鎖定 (Locked)」狀態且保護期已過，維護費會增加 **20%**。

### 3. 土地產權與擴張
* **國產署未售地**：開局除家族地塊外，其餘均為國產署地塊，玩家需以地價 **1.5 倍** 進行初次購買。
* **強制收購**：可對非鎖定的 AI 土地發起收購，但需支付市場溢價。
* **產權鎖定**：支付費用可鎖定土地，防止對手強行買入。

## 🛠️ 技術細節
* **語言**：純 JavaScript (Vanilla JS), HTML5, CSS3。
* **渲染機制**：使用 **CSS Grid** 動態渲染 10x10 網格，並透過 **CSS Variables** (`--f`) 即時呈現建築外觀高度。
* **狀態管理**：核心資料儲存於全域 `city` 物件，每回合動態計算收益、AI 決策與租約變動。

## 🚀 快速開始
1.  複製此專案到本地。
2.  直接使用瀏覽器開啟 `10x10.html` 即可開始遊戲。
3.  **操作目標**：在有限的初始資金下，收購具備 3 路優勢的地塊並進行合併開發，達成資產極大化。
